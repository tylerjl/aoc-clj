# aoc-clj

[Advent of Code][aoc] solutions written in Clojure.

**Warning**: written by a lisp and Clojure novice.

## Usage

Prerequisites
- leiningen

```
$ lein run -- --help
$ lein run -- day1 part1 resources/day1
```

## Testing

```
$ make test
```
